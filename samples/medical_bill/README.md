# Medical Bill Samples / 医療請求サンプル / Factures médicales

- **JA:** `--domain medical_bill` または `medical_bill_jp` を指定。保険者名・診療日・点数が含まれるとフィルタが強化されます。
- **EN:** Run with `--domain medical_bill` for hospital invoices; insurance, patient and procedure fields are extracted into the RAG bundle.
- **FR:** Utilisez `--domain medical_bill` pour les factures médicales ; assurance, patient et actes sont exportés pour la RAG.
