# Invoice Samples / 請求書サンプル / Exemples de factures

- **JA:** `--domain invoice` または `--domain invoice_en` を指定し、請求書 PDF/PNG を配置してください。Date 欄が空でもゲートは通過できるようになっています。
- **EN:** Drop invoice PDFs/PNGs here and run with `--domain invoice` (or `invoice_en`). The gate treats dates as optional so minimalist invoices still pass.
- **FR:** Déposez vos factures (PDF/PNG) et utilisez `--domain invoice` ou `invoice_fr`. La détection accepte l'absence de date.
