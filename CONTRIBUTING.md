# Contributing / 貢献

Thanks for your interest in improving Z-OCR.

## Development quickstart

- Python: 3.10+
- Install (editable):
  - `python3 -m pip install -e ".[dev,api]"`
- Run tests:
  - `python3 -m pytest`

## Coding guidelines

- Keep changes focused and well-scoped.
- Prefer small, reviewable PRs.
- Add/adjust tests when changing behavior.

## Licensing of contributions (important)

Z-OCR is **dual-licensed** (AGPL + commercial). To keep commercial licensing possible:

- By default, contributions are accepted under **AGPL-3.0-or-later**.
- For **substantial** contributions, maintainers may require a **CLA** (Contributor License Agreement) or an explicit additional license grant before merging. See `CLA.md`.

If you are contributing on behalf of your employer, make sure you have the right to do so.
