# Security Policy / セキュリティ

## Reporting a vulnerability

Please do **not** file public issues for security vulnerabilities.

Instead, contact the maintainers privately with:

- A description of the issue and impact
- Reproduction steps / PoC (if available)
- Affected versions/commit (if known)

Contact: kishkavsesvit@icloud.com

## Supported versions

Security fixes are provided on a best-effort basis for the latest released version.
